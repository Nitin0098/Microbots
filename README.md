# Microbots
Microbots are microscale robots capable of forming a magnetic bond between eachother allowing them to create any 3D structure, This repo holds some white papers and preprints with designs of microbots.

This is a theoretical concept with designs, although this design faces some efficiency and energy issues at microscale, but at sub-millimeter scale this design has been mathematically and physically proven to work.
